# 使用
    npm install -g create-app
    然后在控制台使用 create app 或 capp 命令创建项目
    也可以 create app <项目名称> --temolate=react-ts | react | react-labmai

# 模版
    - express
        node + express模版
    - vite-plugin
        开发vite—plugin模版
